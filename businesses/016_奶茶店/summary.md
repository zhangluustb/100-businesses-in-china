# 奶茶店：做了的人告诉你真相——别被"轻松创业"忽悠了

> 你想开奶茶店？觉得一杯奶茶成本2块卖15块，毛利率60%多，听起来很美？但一年倒闭近20万家、有人开3个月就关门、3个审计师合伙半年赔60万——这些数字才是真相。奶茶店不是"轻松创业"，是竞争最惨烈的战场之一。

---

## 一、先说真相：奶茶店是创业"黑洞"

红餐网的数据很扎心：**从2023年11月到2024年11月的13个月内，19.7万家奶茶店闭店倒下了。** 而同一时间新开店127752家，净增长是-17777家——意味着开出来的店比倒掉的少得多。

社交媒体上，奶茶店倒闭分享帖成了一种"流量密码"，大家轮流来复盘自己怎么亏的。店主给出的答案一个比一个扎心：
- 有人3个月就关了门
- 有人半年亏损18万
- 有人经营一年多倒亏30万
- 有人加盟一线品牌开3家店累计亏200多万
- 最夸张的：开业1个月就倒闭，二手设备回收商去收时设备还有九成新

**能撑一年半载的奶茶店，已经算"高寿"了。**

业内人士一句话总结：「茶饮行业已经没有市场增量了」。

> **一句话总结**：奶茶店不是轻松创业，而是竞争最惨烈的"黑洞"。你进场之前，先看看19.7万家倒闭的数据。

---

## 二、这笔账到底怎么算

奶茶行业毛利率60%-70%——这是食材层面的数字，听起来很诱人。一杯奶茶食材成本3-5元，卖15元，毛利率确实不错。

但真实利润≠毛利率。来算算完整账：

| 费用项 | 大约占比 |
|--------|---------|
| 食材成本 | 30%-40% |
| 门店租金 | 15%-25% |
| 人工成本 | 10%-20% |
| 外卖平台抽成+推广费 | 20%-25% |
| 水电+杂费 | 5%-10% |
| **合计** | **80%-120%** |

你没看错，总成本可能超过100%。**做外卖几乎"卖一杯亏一杯"**——这是多个店主共同发现的真相。

外卖平台抽成约20%-25%，加上线上推广费用，到手营收极低。长沙恒星3个月倒闭的咖啡店就是这么回事：定价20元一杯，外卖打折15元，再扣掉平台抽成——到手可能不到10元，还不算食材和人工成本。

3个审计师合伙开奶茶店，蹲点时算出"杭州一点点一分钟出一杯，日营业额7000元，年赚100万没问题"——但这是在节假日蹲点得出的数据，根本不能代表全年常态。

> **加料才是利润点**：基础款利润薄，加芝士、加珍珠、加水果才是利润大头。和鸡蛋灌饼一样——加料比基础款利润率高得多。

## 三、启动要花多少钱

奶茶店启动资金范围很大，取决于你选什么模式：

### 自创品牌（最便宜但最难）

| 项目 | 金额 |
|------|------|
| 小铺位租金（月） | 1500-6000元 |
| 首期租金+押金 | 6000-36000元 |
| 装修 | 3-6万 |
| 设备（制冰机、封口机、冷藏柜等） | 3-8万 |
| 首批食材+包装 | 1-2万 |
| 证照办理 | 0.5-1万 |
| **总计** | **10-30万** |

### 加盟不同品牌

| 品牌 | 加盟费 | 总投资 |
|------|--------|--------|
| 蜜雪冰城 | 7000-11000元/年 | 约30万起 |
| 沪上阿姨 | - | 约27.5万 |
| 古茗 | 首年0加盟费 | 约35万 |
| 茶百道 | - | 约35万 |
| 奈雪的茶 | - | 58万起 |

**注意**：加盟费只是表面数字。蜜雪冰城加盟费只有7000-11000元/年——看起来很便宜。但加上设备（蜜雪冰城要求从总部采购）、装修、首批物料等，总投资30万起步。品牌赚的不是加盟费，是卖给你的原料和设备——蜜雪冰城97%的收入来自"商品和设备销售"。

3个审计师的教训更扎心：步行街12平方铺位，年租金20万+转让费6万+中介费2万+装修6万=34万光铺位就花了34万。还没开业就花了大半预算。

> **别只看加盟费**：蜜雪冰城7000块加盟费很诱人，但实际总投资30万+。品牌赚的是供应链的钱，不是加盟费的钱。

## 四、选址：奶茶店的命门，好位置轮不上外行

选址决定了奶茶店80%的命运。但三个审计师发现了一个残酷的现实：**好位置根本轮不上外行**。

步行街最火的铺位，刚有转让消息就被内部消化了，外人根本看不到。他们花了2万中介费才租到一个铺位，结果发现租金比周围都高——很可能被中介赚了差价。

更惨的是，步行街晚上人最多的时候，小吃街的集装箱摊位挡住了他们80%的人流。眼巴巴看着台阶下人潮涌动，自己的店门可罗雀。

奶茶店选址的铁律：
- **商圈入口/地铁口**：随手买一杯的冲动消费才是奶茶的核心场景
- **学校周边**：学生消费频次高，但客单价低
- **社区门口**：老客稳定但流量有限
- **远离商圈1公里**：基本只有附近住户路过，不符合"随手买一杯"的设定

选址的隐性成本：
- 奶茶行业铺位竞争激烈，好位置被连锁品牌和餐饮老手提前锁定
- 中介费2-5万不等
- 转让费几万到十几万不等
- **一个奶茶店的租金压力是生死线**

## 五、奶茶店为什么竞争这么惨烈

奶茶行业有多卷？来看数据：

**1. 门店数量爆炸**
- 蜜雪冰城4.5万+家店
- 古茗近万家店
- 沪上阿姨8000+家店
- 茶百道8000+家店
- 2023年上半年多家品牌喊出"万店目标"，一年新增3000+家门店

**2. 产品严重同质化**
小红书上兴起"在家自制XX奶茶店招牌产品"热潮——一点点的茉莉奶绿、茶百道的泥石流都能在家DIY。**奶茶店原材料、配方、品控没有太多秘密和门槛**。

口味做不出差异化，消费者忠诚度无从谈起，只能拼价格。

**3. 价格战恶性循环**
- 2021年至今，近30个连锁奶茶品牌客单价出现下滑
- 奈雪客单价3年下降10.7元
- 喜茶客单价3年下降10.4元
- 中腰部品牌常规操作：价格打到10元内
- 越来越多品牌直逼蜜雪冰城的10元以下价格带

**4. 大品牌下沉挤压小店**
- 一线品牌在县城甚至乡镇安营扎寨
- 个体小店知名度、营销能力都欠缺
- 相似口味+相似价位，小店很难抢到大品牌生意
- **蜜雪冰城57.2%门店在三线及以下城市**

一句话总结：**大规模拓店→供过于求→价格战→利润枯竭→小店被挤压→恶性循环**。

## 六、加盟的坑：二线品牌最危险

栗子加盟了三线茶饮品牌，投入40万，存活不到6个月。她的教训是：**加盟开奶茶店，选择不上不下的二线品牌是"最危险"的——要么选最便宜的，要么选大牌。**

三线品牌的坑：
- 品牌认知极低，几乎没有自带客流
- 加盟管理粗放，所谓的运营策略只有低价活动
- 品牌推广完全不上心
- 总部信誓旦旦说"各种帮扶"，实际上什么都没有

一线品牌也不是好选择：
- 冰冰加盟一线品牌开3家店，累计亏200多万
- 茶百道前加盟商称开店亏了30万，评论区众人附和
- 二手平台上茶饮设备卖疯了，回收商一天最多接到13个撤店电话

最残酷的现实：**卖不好亏百万自生自灭，卖得好被品牌在旁边放加盟或直营**。品牌和加盟商根本不在一条战线上。

蜜雪冰城数据里最讽刺的一幕：加盟商做得好→品牌推荐旁边新点位→原来的店客流被分走→两家一起挣扎。

> **加盟的核心矛盾**：品牌要规模（铺更多店），加盟商要利润（少竞争）。这两件事天然冲突。

## 七、合伙开奶茶店的惨痛教训

3个审计师合伙开奶茶店，半年倒闭赔60万。5大失败原因值得所有想合伙开店的人看：

**1. 出于信任合伙，没有股权划分**
每人出资15万，股权平均，没人管事。店长干了3周就受不了要换人，6个月换了5个店长。没人出方案，出了方案也没人执行。

**2. 钱省错了地方**
最该花钱的产品研发、技术上省了——找了熟人给配方，以为省了几万加盟费。结果出品口感全靠运气，**产品研发不是一张配方的事情**。同品牌原料不同产地味道就天壤之别，茶汤水温、时间、风味需要一点点调试。

**3. 蹲点数据不靠谱**
在节假日蹲点数人流，得出"日营业额7000元"的大喜数据。但节假日是奶茶店生意最好的时候，这个数据根本不足以支撑全年利润预估。

**4. 被中介坑选址**
花2万中介费租铺位，后发现租金比周围都高，还被小吃街挡住80%人流。

**5. 赔钱又赔感情**
开店前觉得最惨无非赔钱；开店后发现赔掉了十几年的朋友情份。

> **合伙铁律**：一定要有一个人说了算，签书面协议（出资比例、职责分工、退出机制、违约条款）。

## 八、外卖困局：不卖没销量，卖就得亏钱

恒星3个月倒闭的咖啡店和naomi半年挣扎的社区咖啡店都发现同一个问题：**做外卖几乎卖一杯亏一杯**。

外卖困境：
- 平台抽成20%-25%
- 线上推广费用（竞价排名、满减补贴）
- 外卖配送费补贴
- 包装成本增加
- 餐品摇晃洒出来→需要封口机→合伙人不同意买→吵架→散伙

恒星算账：定价20元，外卖打折15元，扣平台抽成到手10元左右，再扣食材成本3-5元、人工2-3元、包装1元——到手可能还亏。

naomi更惨：社区咖啡店日营业额飘忽不定，多的时候200多元，少的时候几十元，**最差的一天只有1个客人18元**。

> **外卖真相**：不卖没销量（没人知道你），卖就得亏钱（平台吃掉利润）。两条路都是坑。

## 九、必须避开的6个坑

### ❌坑1：在节假日蹲点数人流决定选址

节假日奶茶生意最好，数据根本不能代表全年。一定要在工作日蹲点，看真实日流量。

### ❌坑2：只看加盟费不看总投资

蜜雪冰城加盟费7000块，但总投资30万+。加盟费只是冰山一角。

### ❌坑3：加盟不上不下的二线品牌

要么选最便宜的蜜雪冰城，要么选有号召力的大品牌。中间地带最危险——品牌认知低+管理粗放=双重风险。

### ❌坑4：合伙没协议

出资比例、职责分工、退出机制、违约条款——全部写书面协议。一定要有一个人说了算。

### ❌坑5：在外卖上投入太多精力

做外卖几乎卖一杯亏一杯。前期应该以堂食+外带为主，外卖只是辅助。

### ❌坑6：靠一张配方就开业

产品研发不是一张配方的事情。同样的原料不同品牌味道大相径庭，需要大量调试。没有专业技术人员就开业，出品口感全靠运气。

## 十、奶茶店还有机会吗？看清三条路

奶茶行业已经杀成一片红海，但也不是完全没机会。看清三条路：

### 路A：加盟蜜雪冰城（最稳但利润薄）

蜜雪冰城是目前最稳的选择——4.5万+门店、品牌认知强、供应链完善。但单杯均价6.34元且在下降，日均GMV4184元增速放缓。**蜜雪冰城赚的是供应链的钱，加盟商赚的是辛苦钱**。

适合人群：小城市创业者，有30万启动资金，愿意亲力亲为。

### 路B：自创品牌做差异化（最难但最有想象力）

奶茶产品严重同质化，但差异化仍有空间：
- 做地方特色（比如成都的冰粉奶茶、广东的糖水奶茶）
- 做健康概念（零糖、低卡、功能茶饮）
- 做场景创新（奶茶+自习空间、奶茶+宠物友好）
- 做社区小店模式（不做网红，做老客）

适合人群：有行业经验、懂产品研发、能亲力亲为的人。

### 路C：不做奶茶，做其他饮品赛道

奶茶是最惨烈的战场，但饮品品类里还有其他选择：
- 柠檬茶：南方市场有增长空间
- 鲜榨果汁：健康概念有潜力
- 冰粉/凉虾：低成本摆摊模式
- 咖啡小店：独立精品咖啡有差异化空间

适合人群：想入饮品赛道但不想杀入奶茶红海的人。

> **关键判断**：奶茶的黄金年代已经过去。2027年市场规模达5118亿听着不错，但增量主要被头部品牌吃走。留给独立小店和普通加盟商的空间越来越小。

## 十一、最后说一句最重要的话

> **"奶茶店是创业门槛最低的饮品品类——10万就能启动。但门槛低≠好赚。一年倒闭19.7万家、3个月关门、半年赔60万——这是最惨烈的战场之一。如果你没有行业经验、没有差异化产品、没有好位置，千万别进场。"**

红餐网说的最直接：「任何时候，开一家茶饮店都不是一门稳赚不赔的生意，以后要赚钱也只会越来越难。如果没有做好准备，千万不要盲目入场。」

三个审计师说的最扎心：「餐饮是个能赚钱的行业，但不是想象中能赚快钱的行业。要是坐在吧台刷着剧就能把钱赚到，那些真正热爱、亲力亲为、用心经营的人早暴富了。」

加盟商说的最残酷：「卖不好亏百万自生自灭，卖得好被品牌在旁边放加盟或直营。」

二手设备回收商说的最讽刺：「设备根本收不完。某些品牌的设备我们都不收了，只能当废品处理。」

想干就干，但先看完这19.7万家倒闭的数据再决定。知道真相再出发，你才不会成为下一个3个月关门的人。
