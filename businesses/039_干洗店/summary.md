# 干洗店：干了7年的人告诉你真相

> 你想开干洗店？先看看那些真干过的人说了什么再说。毛利率60%-80%听着很美，但扣完房租、设备折旧、人工，净利可能比打工还少。旺季月入10万很爽，淡季月入2万差点关门——这生意不是躺赚，是把命押在季节上的辛苦活。

---

## 一、先说真相：倒闭率99%不是吓你

小红书上有个干洗店创始人冷总，直接说**"市面上干洗店的倒闭率99%"**。这数字听着夸张，但北京一家7年加盟店的数据告诉你——还真差不多。

为什么倒闭率这么高？三个原因：
- **淡旺季太极端**：1-2月旺季月入10万，夏天淡季月入2-3万，8个月都在苦熬
- **房租是吸血鬼**：一线城市20-30平小店，年租金15-25万，每天睁眼就是400-700元的房租压力
- **加盟坑太多**：加盟费+品牌使用费8-9万，很多品牌收了钱就不管你了

有个博主叫"糖糖想有米"，标题就直说——**"穷人真的不能碰家政干洗店！"**。穷人为什么不能碰？因为你需要：
- 5-20万启动资金
- 一年35-40万的运营成本
- 至少撑过2-3个淡季才能稳定

手头只有几万块的人，开干洗店就是把钱往水里扔。

> **一句话总结**：干洗店不是穷人能碰的生意。你得有钱扛过淡季，有技术洗好衣服，有耐心积累回头客。

---

## 二、这笔账到底怎么算

刺猬公社实地调查了一个北京朝阳区7年加盟干洗店，给出了最真实的成本拆解：

### 开店成本拆解（一线城市）

| 项目 | 金额 |
|------|------|
| 年租金（20-30平） | 15-25万 |
| 设备+装修 | 5-6万 |
| 加盟费+品牌使用费 | 8-9万 |
| 年洗涤耗材费 | 2-5万 |
| 人工+水电等 | 另算 |
| **年总支出** | **35-40万以上** |

算一下保本线：年支出40万，每月3.3万。旺季10万减3.3万=净赚6.7万，淡季2万减3.3万=**亏损1.3万**。一年下来，旺季赚的钱补淡季亏的钱，可能净利也就3-5万。

> **别被毛利率骗了**：干洗服务毛利率60%-80%没错，但那是扣掉洗涤耗材后的毛利率。房租、人工、设备折旧一分没扣。扣完之后，净利率可能只有5%-15%。

### 一件衣服赚多少钱？

北京那家7年店的数据：
- 工作日每天收15件
- 周末每天收30-50件
- 一件衣服洗费平均30-80元
- 鞋子：会员25元/双，非会员50元/双

一天收15件×50元=750元营收。扣掉房租400元+人工200元+耗材50元=到手只有100元。

> **赚钱靠旺季和回头客**：7年老店微信里有2000+回头客，淡季靠这些熟客撑着。新店没有回头客，淡季可能直接关门。

## 三、启动要花多少钱

5-20万启动资金，看你是加盟还是独立开店。

### 方案A：加盟品牌店（10-20万）

| 项目 | 金额 |
|------|------|
| 加盟费 | 3-5万 |
| 品牌使用费 | 2-4万/年 |
| 设备采购（品牌指定） | 5-8万 |
| 装修 | 2-3万 |
| 首批耗材 | 0.5-1万 |
| **总计** | **12-20万** |

加盟的好处：品牌背书、技术培训、总部运营支持
加盟的坑：**很多品牌收了加盟费就不管你**，设备指定供应商比市场贵30%-50%，品牌使用费年年收

### 方案B：独立开店（5-10万）

| 项目 | 金额 |
|------|------|
| 二手干洗机 | 2-3万 |
| 其他设备（熨烫台等） | 1-2万 |
| 装修（简装） | 1-2万 |
| 首批耗材 | 0.5-1万 |
| **总计** | **4.5-8万** |

独立开店的好处：没有加盟费束缚，设备自己选
独立开店的难：没品牌背书，技术全靠自己学，获客更难

> **建议**：新手可以先去干洗店打工3-6个月，学手艺+学经营。然后独立开店，省掉加盟费。如果你非要加盟，一定选有2000+门店的大品牌，别选只有十几家店的小牌子。

## 四、选址：决定80%的生死

### 哪些位置能赚钱

- **高档社区门口**：居民有钱，冬天大衣羽绒服洗得多，客单价高
- **写字楼附近**：西装衬衫干洗需求稳定
- **医院附近**：医护人员白大褂定期洗
- **学校附近**：宿舍区，但客单价低

### 哪些位置不能选

- **菜市场旁边**：油烟味重，衣服还没洗就沾味
- **偏僻小巷**：没客流，连路过的人都没有
- **租金超过2万/月的门面**：每天600+房租，保本线太高

### 北京7年店选址案例

北京朝阳区那家7年店，选在**主干双向路上，辐射3公里居民区**。位置不算最顶级，但7年积累2000+熟客后，淡季也能撑住。

> **选址铁律**：选高档社区门口，租金不超过1.5万/月。别贪大面积——20-30平就够了。

## 五、手艺：干洗店的三个命门

### 命门一：分面料

干洗店最忌讳的错——**把不该干洗的衣服干洗了**。

北京7年店的马姐说：「有的衣服是没办法干洗的」。羊绒可以干洗，但很多面料干洗反而加速老化。她店里的做法：
- **运动鞋、皮包**：手洗，配不同比例洗剂
- **羽绒服、棉服**：机器水洗烘干
- **羊绒**：才是真正干洗
- 干洗机三个月才换一次洗剂

新手最大的坑：客人拿件"干洗"的衣服来，你真干洗了，结果衣服缩水变形——赔钱+丢客。

### 命门二：洗前检查

马姐亲历事故：客人衣服沾油渍，在家用洗洁精预处理后送来，结果和店内洗涤剂反应，**衣服烧出一个大洞**。

从那以后：凡是有东西湿漉漉送过来的，都得特别问清楚。

> **铁律**：洗前必须当面检查衣物，和客人确认洗涤方式。有污渍的先问怎么沾的，预处理过的必须问用了什么。这一步省不得。

### 命门三：洗鞋手艺

马姐洗鞋依然手洗，每双15分钟。刷鞋机太粗暴，对鞋子损伤大。

> **建议**：先学会看面料、分洗涤方式，再去学设备操作。洗坏一件大衣赔几百上千，比洗好一百件衣服赚的钱还多。

## 六、每天的真实作息

干洗店不是8小时上班那么简单：

| 时间 | 干什么 |
|------|--------|
| 早上8:00 | 开门，接收送洗衣物 |
| 上午 | 检查分类、分面料、确定洗涤方式 |
| 下午 | 洗涤、烘干、熨烫 |
| 傍晚 | 归还洗净衣物，接待晚来的客人 |
| 晚上 | 维护设备、补充耗材 |

旺季（冬天）忙到连吃饭时间都没有——每天30-50件衣服排队洗。
淡季（夏天）闲到刷手机——一天15件，收入只够付房租。

> **真相**：干洗店最难受的不是忙，是闲。8个月淡季每一天都在煎熬——房租在涨、人工在付、收入在跌。

## 七、潜规则

### 潜规则一：加盟=收钱不管

知乎上有人说：「中国的加盟，就是骗你的钱」——这话对干洗店行业特别适用。很多干洗品牌：
- 收了加盟费3-5万
- 指定设备供应商（比市场贵30%-50%）
- 品牌使用费年年收
- 技术培训敷衍
- 运营支持=发个手册

> 小红书上"赞哥生意圈"专门做了视频："干洗店投入多少？还能做吗？"——答案是：加盟投入多，独立投入少，但技术门槛高。

### 潜规则二：会员储值=锁客套路

非会员洗鞋50元/双，会员25元/双——差距这么大，当然要储值成为会员。但储值本质是**提前把钱锁在你店里**，如果店倒闭了，会员的钱就没了。

### 潜规则三：很多衣服根本不需要"干洗"

"干洗"听起来高级，但实际大部分衣服是**水洗**——干洗机用的特殊洗剂，三个月才换一次，成本不高。真正干洗的只有羊绒等极少数面料。

客人拿件普通棉质T恤来"干洗"，你用水洗烘干就好了——客人满意、你省钱、衣服也没损伤。

### 潜规则四：线上洗衣平台不是敌人

刺猬公社调查发现：线上洗衣平台（大鲸洗、衣卫士等）看似低价高效，实际投诉量巨大——"暴力洗衣、染色、掉色、变形、丢失"。黑猫投诉近千条。

实体干洗店的优势恰恰是线上平台的劣势：**当面沟通、手洗精细、不伤衣物**。7年店马姐说："我们小店过得挺好的，收入也还可以"——她不怕线上平台抢生意。

### 潜规则五：淡季生存靠回头客

2000+微信回头客撑起了7年店的淡季。新店没回头客怎么办？靠储值会员、靠上门取送服务、靠洗鞋（夏天洗鞋需求稳定）。

## 八、必须避开的坑

### ❌坑1：花大钱加盟小品牌

加盟费3-5万+品牌使用费年年收+设备指定贵30%-50%=**至少亏8万起步**。选加盟品牌要看门店数量——全国2000+门店的靠谱，只有十几家的别碰。

### ❌坑2：没学手艺就开店

洗坏一件大衣赔几百上千。先去干洗店打工3-6个月，学会看面料、分洗涤方式、操作设备，再考虑自己开。

### ❌坑3：选大面积高租金门面

20-30平就够了。租金不超过1.5万/月。面积大了租金高，但收衣量不会因为面积大而增加——客人不会因为你店大就多洗几件衣服。

### ❌坑4：不跟客人当面确认洗涤方式

马姐的教训：客人预处理过的衣服和店里洗涤剂反应，烧出大洞。当面检查、当面确认——这一步省不得。

### ❌坑5：不储备淡季生存资金

旺季赚的钱别全花。存够至少3个月的房租+人工=4-5万，撑过第一个淡季。很多新店就是第一个夏天撑不住倒闭的。

### ❌坑6：只做干洗不做洗鞋洗包

夏天干洗需求极低，但洗鞋洗包需求稳定。不叠加洗鞋业务，8个月淡季可能亏死。

## 九、趋势信号

### 信号1：线上洗衣平台萎缩

O2O洗衣平台近10年没跑出巨头。e袋洗拿了腾讯投资后逐渐衰退，大鲸洗、衣卫士投诉量巨大。**线上洗衣没有取代实体店**，反而帮实体店教育了市场——让更多人开始送洗衣服。

### 信号2：洗鞋成为新增长点

小红书上"十元洗鞋内幕"26赞——洗鞋是一个低客单价但高频的生意。夏天干洗需求低，洗鞋需求稳定，可以补淡季营收缺口。

### 信号3：社区店+线上获客

实体干洗店也在学新玩法——"总部线上运营，达人探店"、上门取送服务、会员储值小程序。**不学线上获客的社区店会被淘汰**。

### 信号4：专业细分趋势

奢侈品洗护、皮包养护、运动鞋精洗——细分品类客单价更高。7年店马姐洗鞋手洗15分钟/双，精洗客单价50-80元/双——比普通干洗利润率高。

## 十、进阶路线

### 第1-3个月：学手艺期

- 去干洗店打工3-6个月，学会看面料、分洗涤方式、操作设备
- 不花钱开店，先花钱学手艺
- **必做**：重点学洗鞋和皮包养护——淡季救命技能

### 第3-6个月：筹备期

- 选址：高档社区门口，20-30平，租金不超过1.5万/月
- 买设备：二手干洗机2-3万+熨烫台1万，比加盟指定供应商省30%
- 开小程序：会员储值+线上预约+上门取送
- **目标**：总投资控制在8-10万

### 第6-12个月：生存期

- 开业迎旺季（秋冬开业最佳）
- 鼓励储值会员锁客
- 叠加洗鞋洗包业务
- **目标**：旺季月营收5-8万，净利1-2万

### 12个月以后：两条路

- **路A**：继续单店经营，积累回头客到2000+，淡季靠熟客撑住
- **路B**：升级为洗护工作室——奢侈品洗护+皮包养护+运动鞋精洗，客单价翻倍

> 北京7年店的成功路径值得参考：加盟→积累2000+回头客→手洗精洗→淡季靠熟客→小程序线上获客→活下来了。

## 十一、最后说一句最重要的话

> **"干洗店是毛利率高但净利率低的生意——毛利率60%-80%，扣完房租人工可能只剩5%-15%。旺季月入10万，淡季月入2万差点关门。穷人别碰，有钱人要学手艺再入场，选址定生死，回头客保命。"**

7年店的马姐说得最接地气：「我们小店过得挺好的，收入也还可以」——但这7年的背后，是2000+回头客、手洗精洗15分钟/双、当面沟通确认洗涤方式的坚持。

干洗店不是躺赚生意，是需要手艺、需要耐心、需要扛过8个月淡季的辛苦活。想干就干，别被"暴利"忽悠了。知道真相再出发，你才不会像99%的人一样倒闭。
