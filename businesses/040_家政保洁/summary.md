# 家政保洁公司：干了10年的人告诉你真相

> 你想开家政保洁公司？先看看那些真干过的人说了什么再说。3万块就能启动听起来很爽，但一个人干就是把自己当机器，雇人干就是把自己当HR——这生意不是"躺赚"，是用体力和管理能力换钱的辛苦活。家庭保洁赛道有人说已经"结束了"，但深度保洁、开荒保洁、专项清洗还有机会。

---

## 一、先说真相：3万启动≠3万赚钱

小红书上"小白不懂家政"说**"三万如何启动一家家政公司"**——确实3万就能启动，但启动≠赚钱。

为什么3万能启动？
- 不需要店面（大部分家政公司是"云店"模式）
- 不需要大设备（保洁工具几千块就够）
- 不需要大量库存（服务为主，没有商品成本）

但3万启动≠3万赚钱：
- 你得有客源，没客源3万块花光就关门
- 你得有人干活，没人干活就只能自己干
- 你得有管理能力，雇人干活但管不好人就是灾难

小红书上"慢品人生"发帖说**"感谢各位网友，今晚决定了，创业项目做家政"**，287条评论里大量从业者分享真实经验。最高赞的一条核心观点：**家政不是门槛低的生意，是看起来门槛低实际上很专业的生意。**

> **一句话总结**：家政保洁启动门槛低，但赚钱门槛高。3万能启动，但至少1年才能稳定赚钱。

## 二、这笔账到底怎么算

### 家政保洁的成本结构

家政保洁是轻资产生意，最大的成本不是设备，是**人工**。

| 成本项 | 金额/占比 |
|------|------|
| 保洁工具+耗材 | 2000-5000元（一次性） |
| 小程序/获客渠道 | 500-2000元/月 |
| 人工成本 | 占营收50%-70% |
| 交通成本 | 占营收5%-10% |
| 其他运营 | 占营收5%-10% |
| **净利润率** | **15%-30%** |

商务部数据：家政服务企业平均收入费用率22%，意味着**78%是可变成本**（主要是人工）。

### 一个人干 vs 雇人干

#### 一个人干（月入5000-15000）

| 项目 | 金额 |
|------|------|
| 日均接单 | 2-3单 |
| 每单收费 | 150-300元（3-5小时） |
| 日营收 | 300-900元 |
| 月营收 | 9000-27000元 |
| 扣除交通+耗材 | 到手70%-85% |
| **月净利润** | **6300-22950元** |

小红书"一只幸运猫"说：**一个人开了个家政公司，是真的爽啊！**——一个人干的好处：全部利润归自己，不用管人，时间自由。但一个人干每天最多3单，体力有限，生病就没收入。

#### 雇人干（月入1万-5万+）

| 项目 | 金额 |
|------|------|
| 雇3-5个阿姨 | 每人3000-5000元/月 |
| 每人日接单 | 2-3单 |
| 每单收费 | 150-300元 |
| 月总营收 | 27000-45000元 |
| 扣除人工 | 剩余30%-50% |
| 扣除其他 | 再扣5%-10% |
| **月净利润** | **5400-18000元** |

雇人干的难点：**管人是最头疼的**。阿姨迟到、偷懒、和客户吵架、辞职——每一个问题都直接影响营收。

> **选择建议**：起步阶段一个人干3-6个月，积累客源和经验。月营收稳定在1.5万以上时，开始雇1-2个阿姨帮忙。

## 三、启动要花多少钱

2-10万启动资金，看你是"云店模式"还是"实体店模式"。

### 方案A：云店模式（2-3万，最推荐）

| 项目 | 金额 |
|------|------|
| 保洁工具套装 | 1000-3000元 |
| 消耗品首批 | 500-1000元 |
| 小程序搭建 | 0-2000元 |
| 营销推广 | 1000-3000元 |
| 交通工具 | 0-3000元（电动车） |
| 营业执照 | 0-500元 |
| **总计** | **2500-12500元** |

云店模式就是：没有实体店面，在家办公，客户线上下单，你或你的阿姨上门服务。3万块确实够启动。

### 方案B：实体门店模式（8-15万）

| 项目 | 金额 |
|------|------|
| 门店租金（半年） | 3-6万 |
| 门店装修 | 1-2万 |
| 设备+工具 | 3000-5000元 |
| 营销推广 | 5000-10000元 |
| 首批人工储备 | 2-3万 |
| **总计** | **8-15万** |

实体门店的好处：有门面增加信任感，方便客户上门咨询。但租金是固定成本，每个月不管赚不赚钱都要付。

> **建议**：新手从云店模式起步，3万以内就能干。别花冤枉钱租门面——家政的客户不来你的门店，你去客户家。

## 四、选址：不需要门店，但需要客户

### 家政保洁不需要选址——你需要的是客源

家政服务和餐饮不一样，客户不来你的门店，你去客户家。所以"选址"的概念变成了"选客源区域"。

### 哪些区域客源多

- **高档小区**：业主有钱，愿意花150-300元/次做深度保洁
- **新交付楼盘**：开荒保洁刚需，200-400元/次
- **写字楼**：办公保洁合同稳定，月费500-2000元
- **医院/学校**：长期合同，但竞争激烈

### 获客方式

小红书上高赞博主鸿鑫姐（588赞528藏223分享）的获客方法：
- **社区微信群**：加入小区业主群，发服务信息
- **美团/58同城**：线上获客，但平台抽成20%-30%
- **抖音/小红书**：发保洁前后对比视频，吸引客户
- **口碑转介绍**：最好的获客方式，零成本，转化率最高

> **获客铁律**：起步靠线上平台，稳定靠口碑转介绍。美团/58同城抽成高但能快速获客，口碑转介绍零成本但需要时间积累。

## 五、手艺：家政保洁的三个命门

### 命门一：清洁效率

一个人干每天最多3单，每单3-5小时。如果你的清洁效率比别人低30%，一天只能接2单——收入直接减少33%。

提升效率的方法：
- 专业工具（蒸汽拖把、高效清洁剂）比普通工具效率高50%
- 流程标准化（先厨房→卫生间→卧室→客厅）减少重复劳动
- 专注技巧（马桶30分钟搞定vs新手1小时）

### 命门二：服务态度

家政是服务行业，态度决定回头客。

小红书"家政十年~赵程羽"（205赞210评235藏）分享了10年店长的经验——核心观点：**态度好+干活细致=回头客不断**。

客户最在意什么？
- 不迟到（迟到5分钟就扣印象分）
- 不偷懒（每角落都擦到位）
- 不碰私人物品（最忌讳翻客户东西）
- 主动沟通（遇到问题先问客户怎么处理）

### 命门三：专项技能

普通保洁单价低（30-50元/小时），但专项保洁单价高：
- **开荒保洁**：200-400元/次（新房交付后的首次清洁）
- **深度保洁**：300-500元/次（油烟机、空调、沙发深层清洁）
- **家电清洗**：单独收费，80-200元/台
- **奢侈品护理**：500-2000元/件

> **建议**：学会普通保洁就够启动，但一定要尽快学开荒保洁和深度保洁——这才是利润大头。

## 六、每天的真实作息

一个人干家政保洁的真实时间表：

| 时间 | 干什么 |
|------|--------|
| 早上8:00 | 出门去第一单（路上30-60分钟） |
| 8:30-12:30 | 第一单保洁（3-4小时） |
| 中午 | 路上赶第二单，吃饭在路上 |
| 13:00-17:00 | 第二单保洁（3-4小时） |
| 晚上 | 处理预约、回复微信、准备第二天 |
| 周末 | 探单更多（2-3单连续干） |

一个人干的辛苦：
- 每天路上2-3小时，不算工作时间但消耗体力
- 干完3单后腰酸背痛——家政是体力活
- 夏天闷热、冬天冷手——户外作业

雇人干的辛苦：
- 不用自己干体力活，但要管人——更累的是心
- 阿姨辞职、客户投诉、单子调配——每天都在处理问题

> **真相**：家政不是轻松的活。一个人干累身体，雇人干累脑子。选哪种累你愿意承受。

## 七、潜规则

### 潜规则一：平台抽成是暗坑

美团/58同城获客快，但抽成20%-30%。你收客户300元，平台抽走60-90元，到手只有210-240元。而且平台要求价格不能太高——低价竞争让你利润更薄。

小红书"momo"吐槽京东家政：**"即使便宜也请大家把京东家政的差评打起来！"**（111赞110评）——平台家政质量差，客户体验不好，反过来也影响你（客户对家政行业信任度下降）。

### 潜规则二：阿姨管理是最大难题

雇阿姨干活，最头疼的问题：
- **迟到**：客户等了半小时阿姨还没到
- **偷懒**：本应3小时完成的只干了2小时
- **辞职**：干了3个月突然说不干了
- **和客户吵架**：一句不当的话就丢客

解决方案：**阿姨按单结算，不按月发工资**。干一单拿一单的钱，迟到扣钱，偷懒扣钱。这样阿姨有动力干好，你也有抓手管理。

### 潜规则三：家庭保洁赛道有人说"结束了"

小红书"老范有话说"视频标题直接说：**"家政创业家庭保洁赛道已经结束了"**。为什么？竞争太激烈了——58同城上家政公司几百上千家，价格战打到30元/小时，利润几乎为零。

但"结束了"不等于"没机会"。普通保洁竞争过度，但：
- 开荒保洁仍有空间（新房交付刚需）
- 深度保洁利润高（油烟机、空调清洗）
- 专项清洗需求增长（家电清洗单独赛道）

### 潜规则四：口碑是你最值钱的资产

家政行业最核心的获客方式不是平台，是**口碑转介绍**。一个满意客户会推荐2-3个新客户，零成本获客。反之，一个不满客户会告诉10个人你服务差。

所以：宁可少接单，也要干好每一单。一单300元做得好→3个转介绍→900元零成本营收。一单300元做得差→10个差评→再也接不到单。

### 潜规则五：储值卡不是必须的

有些家政公司推储值卡（预付5000元享10次）——慎推。家政不像干洗店有稳定需求，客户可能搬家、换工作、需求变化。储值卡退款纠纷多，不如按单收费简单透明。

## 八、必须避开的坑

### ❌坑1：花大钱租门面

家政的客户不来门店，你去客户家。租门面只是增加固定成本。起步用云店模式（在家办公），省掉租金。

### ❌坑2：完全依赖平台获客

美团/58同城抽成20%-30%，而且低价竞争。平台获客是起步手段，不是长期依赖。尽快转型到口碑转介绍+社区群获客。

### ❌坑3：不学专项保洁只做普通保洁

普通保洁30-50元/小时，竞争激烈利润薄。开荒保洁200-400元/次、深度保洁300-500元/次——这些才是赚钱点。

### ❌坑4：雇阿姨按月发固定工资

按月发固定工资=阿姨没有动力干好。按单结算、迟到扣钱、好评奖励——让阿姨的收入和服务质量挂钩。

### ❌坑5：一个人干太久不升级

一个人干月入1-1.5万就到天花板了。达到这个水平时，必须开始雇人。否则体力撑不住、收入不增长、生病就没收入。

### ❌坑6：不做保险和合同

家政服务有风险：阿姨受伤、客户财物损坏。必须买家政服务保险（一年几百块），和客户签服务协议。

小红书上大量一人家政公司分享经验——云店模式、在家办公、线上获客。**不需要门店的家政公司是趋势**。

家电清洗、深度保洁、开荒保洁、奢侈品护理——家政正在从"什么都做"分化为专项赛道。**专项赛道客单价更高、竞争更小**。

京东家政差评111赞110评——平台家政质量差，客户回到找个人/小公司。**个人家政和小团队反而更有优势**——服务好、沟通直接、灵活。

抖音发保洁前后对比视频、小红书分享经验获客、微信群推广——**会用互联网获客的家政公司活得更好**。

## 十、进阶路线

### 第1个月：一个人干试水

- 投资2500-5000元买工具+耗材
- 注册营业执照（个体户即可）
- 加入3-5个社区微信群获客
- 上美团/58同城接单
- **目标**：日均1-2单，月营收3000-6000元

### 第2-3个月：学专项技能

- 学开荒保洁（新房交付后的首次清洁）
- 学深度保洁（油烟机、空调清洗）
- 学家电清洗（单独收费80-200元/台）
- 减少平台依赖，增加口碑获客
- **目标**：日均2-3单，月营收8000-15000元

### 第3-6个月：雇人升级

- 雇1-2个阿姨帮忙（按单结算）
- 建微信群积累熟客
- 专注深度保洁和开荒保洁（利润高的品类）
- **目标**：月总营收1.5-3万，净利5000-10000元

### 6个月以后：两条路

- **路A**：继续小团队模式（3-5人），稳定月入2-5万，轻松管理
- **路B**：升级为公司化运营——10+员工、专项赛道、线上品牌

> 小红书"一只幸运猫"的路径：一个人干→积累客源→觉得很爽→雇人扩大。但记住：一个人干的"爽"是体力换的，雇人后的"爽"是管理换的。
