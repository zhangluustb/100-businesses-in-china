## 一、核心数据来源

### 来源1：即时设计 - 设计师接单攻略：平台推荐与接单赚钱全流程指南
**来源**：即时设计（js.design）
**🔗链接**：[设计师接单攻略：平台推荐与接单赚钱全流程指南](https://js.design/special/article/designers-take-orders.html)

**核心数据**：

**报价阶梯**：
- 刚入行新手：200–500元/单（电商详情页、社团海报）
- 2-3年经验：2000–5000元/单（品牌视觉项目）
- 成熟设计师：1万+/单（品牌全案）

**国外接单平台**：
- Upwork：全球最大自由职业平台，1000万+注册用户，180+国家，内置安全支付系统
- Fiverr：零工经济平台，LOGO/海报短平快项目，可自设价格
- Freelancer：竞标模式，入门门槛低
- Toptal：只接受全球前3%自由职业者，高端客户
- 99designs：设计竞赛型平台，LOGO/包装/网页
- Guru：老牌平台，费用较低
- Coroflot：作品集展示+招聘结合

**国内接单平台**：
- 猪八戒、一品威客、特创意、米画师、特赞、站酷海洛等

**获客渠道**：
- 平台曝光（站酷、小红书、抖音、微博）
- 作品集展示（精挑3-5个代表作）
- 私域渠道（朋友圈、公众号）
- 客户转介绍

**避坑提醒**：
- 警惕某宝/闲鱼超低价单子
- 避免模糊需求+无限改稿
- 正规平台不会要求先交押金

### 来源2：站酷 - 设计师必读-设计接单技巧避坑攻略（张增顺）
**来源**：站酷（zcool.com.cn）
**🔗链接**：[设计师必读-设计接单技巧避坑攻略](https://m.zcool.com.cn/article/ZMTUxNzgyOA==.html)

**核心数据**（二万字干货，十多年设计经验总结）：

**报价参考**：
| 级别 | 海报设计报价 | 备注 |
|------|-------------|------|
| 在校生/刚毕业 | 100-200元/1p | 作品青涩，不够商业 |
| 工作2-3年 | 300-500元/1p | 初步商业设计能力 |
| 工作5-10年+ | 1000-2000元/1p | 资深设计师 |
| 设计大佬 | 电影海报级别 | 价格不菲 |

**报价公式**：
- 日薪 × 工期时间 × 2 = 报价参考价格
- 例：日薪500，工期3天，浮动2 → 3000元参考报价
- 另一种：基本价值 + 设计难度 + 数量 + 工期 + 利润浮动 = 最终报价

**付款方式**：
- 30%+70%（不太建议）
- 50%预付+50%尾款（最常用）
- 50%预付+40%初稿+10%尾款（风险最低）

**接单16步流程**：
1. 需求匹配 → 2. 谈判技巧 → 3. 案例展示 → 4. 需求了解 → 5. 设计报价 → 6. 工期明确 → 7. 初步建议 → 8. 签订合同 → 9. 预付定金 → 10. 创意设计 → 11. 初稿审查 → 12. 修改 → 13. 设计定稿 → 14. 尾款结算 → 15. 完整交付 → 16. 项目结束

**接单渠道**：
1. 签约供图（视觉中国、昵图网、摄图网、包图网、千图网、图虫、熊猫办公、稿定设计、wps稻壳儿、站酷海洛）
2. 接单平台（猪八戒、淘宝、一品威客、特创意、米画师、特赞、圆领）
3. 平台曝光（站酷、小红书、抖音、微博、朋友圈、公众号）
4. 影响力（参加商业大赛）
5. 客户转介绍
6. 信息分发（微信群、QQ群）

**避坑要点**：
- 不做比稿/试稿（或收费比稿）
- 不免费设计（免费不是长久合作模式）
- 修改不超过3次，超出另算费用
- 必须签合同
- 必须收预付款（50%）
- 警惕白嫖骗稿
- 警惕外包再外包
- 警惕甲方审核领导过多
- 警惕不签合同、没预付款的甲方

**版权注意**：
- 字体侵权：使用免费商用字体，或确认授权范围
- 图片素材：不使用来源不可靠的图片
- 素材网站：注意查看授权范围（个人商用/企业商用）

**设计合同6大要点**：
1. 工作内容确定
2. 交付节点+修改范围和次数
3. 知识产权责任认定
4. 付款节点（50%预付+50%尾款）
5. 合同清单/附件
6. 争议管辖约定（原告方所在地）

## 二、UGC金句汇总

- 「免费一定不是长久合作模式。刷一次脸还能第二次吗？」——站酷张增顺
- 「设计农民工」——设计师自嘲，售卖的是时间和创意
- 「花100想做一万效果的甲方」——设计接单常见客户类型
- 「一稿过都是谎言，人家花钱了不多改几版不罢休」——修改的真实状态
- 「低价容易陷入改稿无底洞」——即时设计提醒
- 「前10个客户可能带来的收入有限，但其中的2-3个如果被你维护好，后续的复购和转介绍会远超你想象」——资深设计师经验
- 「没有金刚钻不拦瓷器活，什么段位的接什么段位的单子」——匹配度原则
- 「你报了价格，客户半天没回复，不要着急降价，一定要稳」——报价心理战
- 「别听画饼，好好干后面不会亏待你——就不谈钱的客户，远离吧！」——避坑金句
- 「做PPT的朋友，人家只做PPT，排版就是好看，单子咔咔的多」——聚焦定位的重要性
