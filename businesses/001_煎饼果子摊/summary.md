# 煎饼果子摊：小红书上干了3年的人告诉你真相

> 你想摆摊卖煎饼果子？先看看小红书上那些真干过的人说了什么再说。这里没有"月入3万"的鸡汤，只有踩过的坑、赚过的钱、和那些他们不会告诉你的潜规则。

---

## 一、先说真相：月入3万？别做梦了

小红书上搜"煎饼果子摆摊收入"，出来的第一条是博主「岩岩在厦门」——「每天摆摊2小时能赚多少钱」，265赞。评论区有人羡慕，有人冷静。

**真实数据**：评论区有人说「200不少了，有时0」。摆摊收入不稳定，好的时候日入400，差的时候一天一个都卖不了。

知乎上一个干了3年的摊主说了实话：一天摊150-170个，6块一个，日营业额900-1300元，扣食材到手400-600。**这是小镇老摊主，不是新手。**

**月入3万那个大妈**？在北京黄金地段地铁口，凌晨2点起床备料，夫妻两个人一起干，全年只休初一初二。她挣的是辛苦钱，不是躺着赚。

> **一句话总结**：煎饼果子能养家糊口，但不能让你暴富。想月入3万？那你得把命搭进去。

## 二、这笔账到底怎么算

小红书博主「郑大钱摊煎饼」专门做了一期「摆摊卖杂粮煎饼，一个煎饼的成本和利润分析」，869赞、640收藏——这说明什么？**算账是摆摊人最关心的东西，没有之一。**

### 一个煎饼果子的成本拆解

| 项目 | 成本 |
|------|------|
| 面糊（绿豆面+小麦面） | 约0.5元 |
| 鸡蛋1个 | 约0.6元 |
| 薄脆/油条 | 约0.3元 |
| 酱料+葱花+香菜 | 约0.3元 |
| 火腿肠（加料） | 约0.8元 |
| 燃气+损耗 | 约0.2元 |
| **基础煎饼总成本** | **约2元** |
| **加肠煎饼总成本** | **约2.8元** |

基础卖6-7块（山西网友说「基础款7块」，江西网友说「我这小地7块起一个了」），毛利率60%以上。加蛋加肠加里脊，每个加料1-3元，加料的利润率比基础款更高——**这才是真正的赚钱点**。

### 但评论区的真相

有人问：「**摊位成本呢？**」——博主没算摊位费。好的位置一个月几百到几千，这是隐性成本。

还有人问：「**学费呢**」——暗示有人花钱学摊煎饼，这是另一个隐性成本（别花冤枉钱，后面会讲）。

> **37kr验证**：煎饼果子月入过万只需客流10人/小时，但前提是"不考虑摊位成本"。算上摊位费，月入过万的门槛比你想的高得多。

## 三、启动要花多少钱

**比你想的少得多**。

| 项目 | 金额 |

| 二手三轮车 | 500-800元 |
| 逆变器+扒炉铁板一体机 | 400-600元 |
| 餐具/铲子/保温箱 | 100-200元 |
| 首批食材 | 200-300元 |
| **总计** | **1200-2000元** |

小红书评论区有人问「有没有三轮车转卖的」——二手市场很活跃，不用买新的。

深圳「煎饼王子」的笔记里展示了自己的摊位——灯箱、铁板、三轮车，全套设备看起来很简朴但够用。

> **别花冤枉钱**：有人花5000块买全新电动餐车，没必要。小红书上卖了3年的摊主「郑大钱摊煎饼」用的设备也就几百块。先试水，二手就行，卖不好亏不了几个钱。

## 四、选址：决定了你80%的收入

小红书评论区的真相：

- **「别吹牛逼了，城管不赶你向鸭子。低调一点不好吗？」**——城管是真实威胁，不是吓唬你
- **杭州网友**：「目前在杭州余杭躺平，也想去摆摊了」——摆摊潮导致竞争加剧

### 哪些位置能赚钱

- **地铁口**：早上人流量爆炸，北京通州果园地铁口那个大妈就是这么干的
- **学校门口**：初中高中门口最稳，学生天天来买
- **小区门口/十字路口**：下班回家顺手买
- **工厂门口**：工人早饭刚需

### 选址的铁律

**固定位置固定时间出摊**。知乎17年老摊主孙秀霞的经验：每天准时出摊，老顾客知道你的"上班点"。**60%的营收是回头客贡献的**。

> 你到处跑，老顾客找不到你，回头客就没了。每天重新从零开始。

## 五、手艺：脑子会了手不会

小红书评论区最真实的对话：

网友：「锅温还没好就放面糊了[偷笑R]」
博主「煎饼王子」回复：「怎么可能，没温好的话会粘的[笑哭R]」

**新手最大的坑**：

- **皮厚了** → 影响味道，顾客说"像吃饼"
- **皮薄了** → 翻面碎掉
- **火大了** → 烤焦
- **火小了** → 摊不圆

小红书博主「王周到小吃实训」的商用教程（3082赞、5446收藏）评论区吵成一团：

- **天津网友**：「小麦粉做薄脆你不是开玩笑呢？都是绿豆粉做才好吃。」——正宗天津做法是绿豆面
- **四川网友**：「看到夹肠我就退出了」——传统派不接受加火腿肠
- **网友**：「放的啥东西不说，配比不说，等于啥都没说」——教程不够透明

> **建议**：先在家练30-50个，味道稳定了再出摊。第一个星期出摊可能一天卖不了几个，别慌，这是正常的新手期。

## 七、潜规则：那些小红书上不会置顶告诉你但评论里有人说了的事

### 潜规则一：城管是真的会赶你

小红书杭州卖菜煎饼博主评论区：「别吹牛逼了，城管不赶你向鸭子。低调一点不好吗？」

摆摊晒收入、晒位置 = 同行嫉妒 + 城管关注 = 生意可能更快结束。**做生意要低调。**

### 潜规则二：同行举报是无声的战争

小红书上有个网友分享：自己摊子生意很好，结果被同行举报无证经营，罚款2000元。

### 潜规则三：摆摊培训课是割韭菜

小红书评论区有人说「学费呢」「去哪学啊」——6800块的烤肠培训，到手就是2毛一根的淀粉肠货源+话术模板+烤肠机说明书。**配方网上免费能找到，别花冤枉钱。**

> "直播摆摊的，十个里有八个想当导师，剩下两个被坑惨了在维权。"——小红书评论区共识

### 潴规则四：天气是你的老板

刮风下雨 = 不能出摊 = 当天收入归零。评论「200不少了，有时0」就是最真实的写照。**一年365天，实际能出摊的大概280-300天。**

### 潴规则五：口味之争是真坑

小红书评论区天津网友和四川网友吵起来了：
- 天津正宗派：绿豆面，不加肠
- 全国改良派：加蛋加肠加里脊才是标配

你在哪个城市，就做哪个口味。**别在北京卖天津正宗煎饼果子**，顾客不认。

## 八、必须避开的坑

### ❌坑1：花大钱买全新设备

2000块就能启动，有人花5000买全新电动餐车——**没必要**。小红书上干了3年的「郑大钱摊煎饼」用的就是几百块的二手设备。

### ❌坑2：囤太多食材

新手一周内客流不稳定，少量采购是铁律。「200不少了，有时0」——卖不好就是损耗。

### ❌坑3：不搭配饮品

小红书评论区多人推荐：**绿豆汤、冰镇酸梅汤、豆浆**。卖煎饼搭配饮品，客单价直接翻倍。夏天一杯冰绿豆汤2-3块，成本不到1块。

### ❌坑4：不做回头客生意

有人每天换位置——**错了**。位置固定，老顾客才能找到你。建微信群，发出摊时间，60%营收靠回头客。

### ❌坑5：忽视性价比

6-7块基础款是消费者的心理价位。做价格刺客等于自断客路。

## 十、进阶路线：从试水到稳赚

### 第1个月：试水期

- 投资1200-2000元，二手三轮车+铁板一体机
- 选1-2个位置试出摊
- 练手艺，每天目标50-80个
- **必做**：搭配卖饮品（绿豆汤/酸梅汤），客单价翻倍
- **目标**：日营收300元以上

### 第2-3个月：稳定期

- 固定位置+固定时间
- 建微信群积累熟客
- 优化产品：基础款6-7块+3-5个加料选项
- **目标**：日营收500元以上，回头客占60%

### 第3-6个月：增长期

- 小红书发摆摊日常（真实内容，不搞虚假引流）
- 摊位接地气风格（干净+标价清晰，不要搞文艺风）
- **目标**：月净利润8000元以上

### 6个月以后：两条路

- **路A**：继续摆摊，稳定赚钱，享受自由
- **路B**：升级为门店——不受天气影响、售价可提高、可叠加饮品消费
