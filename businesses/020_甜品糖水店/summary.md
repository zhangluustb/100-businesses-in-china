# 甜品糖水店 - 大白话创业真相

> 投资区间：5-15万 | 行业分类：饮品类 | 关键词：甜品糖水
> 🔗 真实数据来源：36氪/咖门、界面新闻/Tech星球、知乎、小红书UGC

---

## 一、先说真相

甜品糖水店，正在上演"冰火两重天"——连锁品牌疯狂扩张，个体店主疯狂关店。

**真相一**：黑龙江投入21万开糖水店，每天卖不到300元。重庆老板4月开业10月腰斩，周末连500元都难保。小红书上"一天只卖60元，连水电费都不够""16万积蓄全亏光"的帖子到处都是。

**真相二**：糖水是"偶发性解馋消费"，不是刚需。奶茶一个人一杯随时喝，糖水得坐下来用勺子慢慢吃——消费频次根本不一样。糖水店日均客流20人，奶茶店日均客流过百。

**真相三**：连锁品牌降维打击。麦记牛奶2025年新开591家店增速1200%，以前一碗糖水卖25-30元，现在连锁品牌同类只卖18-20元。你开个体糖水店，被连锁品牌价格碾压。

**真相四**：备料极其繁琐。浙江店主每天7:30开工挑莲子、泡薏仁、煮红豆、熬银耳、打芋泥，到11点才备齐——"实在太累了，早知道肯定不开店"。

> **一句话总结**：糖水赛道看似火热，但个体糖水店是"连锁品牌吃肉、个体店主喝风"的格局。消费频次低+备料累+价格战，三重打击。

## 二、这笔账怎么算（成本拆解）

### 街边小店模式成本拆解

| 项目 | 成本 | 说明 |
|------|------|------|
| 房租（押一付三） | 1.5-3万 | 20-30平社区铺位 |
| 装修 | 1-3万 | 简装为主，别搞文艺装修 |
| 设备（冷柜+蒸煮器具） | 1-2万 | 糖水需要冷冻+蒸煮设备 |
| 首批食材+耗材 | 2000-3000元 | 各种豆类、银耳、芋头、糖 |
| 证照+杂费 | 2000元 | |
| **合计** | **5-10万** | **街边小店启动** |

⚠️ 商圈加盟投入远超此数：张远投入90万，苗雨投入73万，黑龙江店主投入21万。**商圈加盟是另一个世界，普通人别碰。**

### 一碗糖水的成本账

| 项目 | 成本 |
|------|------|
| 豆类/银耳/桃胶等底料 | 1-2元 |
| 牛奶/椰奶 | 0.5-1元 |
| 芋圆/木薯等小料 | 0.5-1元 |
| 糖 | 0.3元 |
| PP碗+勺子 | 0.8-1元 |
| 人工（摊到每碗） | 1-2元（备料3.5小时摊到） |
| 房租水电（摊到每碗） | 1-2元 |
| **一碗总成本** | **5-7元** |
| **售价** | **18-28元（正价）/ 16-19元（价格战）** |
| **毛利率** | **70%（正价）/ 50%（外卖后）** |

> 毛利70%看起来很好？但日均客流只有20人！20人×18元=360元日营收，扣掉房租人工，一天净赚可能只有几十块。

## 三、启动要花多少钱

**街边小店启动：5-10万**

- 最省方案5万（社区小铺+简装+二手设备+精简菜单）
- 标准方案8-10万（好位置+品质装修+全套设备）
- 商圈加盟方案21-90万（普通人请远离）

**关键提醒**：
- 糖水店面积需求比奶茶店大——需要堂食区，至少20-30平
- 设备比奶茶店多——冷冻柜+蒸煮器具+备料区
- 至少需要2人（1人备料1人前台），4人更合理

> **建议**：先做私域/社区糖水配送验证需求，日稳定20碗以上再考虑开店。千万别一上来就投商圈加盟。

## 四、选址

### 选址原则

**首选**：社区底商+学校附近
- 赵记传承在广州的选址模式：社区+学校附近街边店
- 家庭亲子群体是糖水核心客群
- 客人坐下来吃，需要堂食区

**次选**：写字楼附近（做下午茶需求）
- 白领下午3-5点可能需要一碗糖水"犒劳自己"
- 但频次远不如咖啡奶茶

**千万别选**：
- ❌ 商场里（投入70-90万，2-3年才可能回本）
- ❌ 瑞幸/奶茶连锁旁边（被价格碾压+客流分流）
- ❌ 纯北方城市冬天（"天一冷，生意直接入冬"）
- ❌ 同区域已有3家以上糖水店（泉州几公里内8家，根本不够分）

### 摆摊模式可行吗？

糖水不太适合摆摊——因为糖水需要坐下来吃，不像奶茶可以边走边喝。但可以做"糖水外卖配送"模式，在小区门口取货。

## 五、手艺/命门

### 核心手艺

1. **糖水出品品质**——每碗都得有"手工感"
   - 桃胶要泡发到位（至少8小时）
   - 银耳要炖出胶质（1-2小时）
   - 红豆要煮到绵密起沙
   - 芋圆要现搓现煮
   - 每一步都耗时耗力——"光是处理物料就得大半天"

2. **备料管理**——SKU繁杂是最大挑战
   - 糖水SKU比奶茶多得多
   - 各种豆类、银耳、桃胶、芋圆、木薯……每种都有不同的泡发/煮制时间
   - 备货少了不够卖，备货多了损耗大
   - **精简SKU是生存关键**：别搞20种糖水，8-10种足够

3. **季节切换**——冬夏菜单差异大
   - 夏天：绿豆沙、冰银耳、杨枝甘露
   - 冬天：姜撞奶、热银耳羹、红豆沙
   - 冬天生意大幅下滑，热饮糖水也难救

### 命门：消费频次

- 糖水是"偶发性解馋消费"，不是刚需
- 咖啡有成瘾性，奶茶适配全场景
- 糖水需要坐下来慢慢吃——场景受限
- 饱腹感强=客户不会天天来
- **这是糖水店最致命的基因缺陷**

## 六、每天的真实作息

**小店模式作息（典型）**：

```
7:30   到店开工备料：挑莲子、泡薏仁、煮红豆、熬银耳、打芋泥
11:00  备料完成，开门营业
12:00  午间短暂高峰（可能5-8碗）
14:00  清闲时段，可能就3-5个客人
15:00  下午茶时段短暂来一波
17:00  晚间家庭客群（亲子消费）
20:00  客流减少
21:30  关门，清洗设备、盘点食材
22:30  回家——明天7:30又得开工备料
```

**最累的不是营业时间，是备料时间**：
- 3.5小时备料 = 一天工作的40%时间花在备料上
- 备料不能偷懒——银耳没炖出胶质、红豆没起沙，客户一口就能尝出来
- 每天重复同样的工序，日复一日

> "实在太累了，早知道肯定不开店"——这是糖水店主最普遍的感受。

## 七、潜规则

1. **糖水"奶茶化"是趋势但不是出路**
   - 茶颜悦色酥山糖水铺已覆盖500家门店
   - 喜茶推出"红豆沙牛乳""桃胶炖奶"
   - 趋势是：糖水不再独立，而是奶茶的一个补充品类
   - 独立糖水铺会被奶茶品牌直接分流

2. **外卖是救命稻草也是利润杀手**
   - 外卖扣点20-25%
   - 但没有外卖，你一天可能就卖360元
   - 外卖半小时蹲点：仅接2单
   - 糖水外卖配送难——碗装比杯装更容易漏洒

3. **加盟糖水铺比加盟奶茶更坑**
   - 糖水SKU更复杂，稽查督导更严格
   - 苗雨的门店"经常因为各类原因被罚钱"
   - 加盟管理难度更高，总部控制更严

4. **糖水"健康养生"是伪命题**
   - 糖水里的糖含量很高——一碗红豆沙含糖20-30g
   - "比奶茶更健康"只是心理安慰
   - 消费者踩过无数消费陷阱后，开始质疑"干净配料表"

## 八、必须避开的坑

### ❌ 坑1：跟风进商圈开糖水铺

张远投入90万在一线城市商场开糖水铺，月销20万扣掉成本根本赚不到钱。苗雨投入73万加盟麦记牛奶，前2月赚10万第3月亏3000元淡季日赚200-300元。**商圈糖水铺是超级加盟商的豪赌游戏，普通人请远离。**

### ❌ 坑2：在北方城市开糖水店

"天一冷，生意直接入冬"。黑龙江投资21万每天卖不到300元。糖水是广东南方特色小吃，北方市场反季经营挑战极大。冬天推出姜撞奶、热银耳也难救——客户更愿意买一杯热奶茶。

### ❌ 坑3：SKU太多备料太复杂

糖水SKU繁杂是最大的运营陷阱。8-10种糖水足够覆盖不同需求。别搞20种SKU——每种都需要不同的泡发/煮制时间，备货少了不够卖多了积压损耗，利润被不断蚕食。

### ❌ 坑4：不做季节菜单切换

夏天只做冰品、冬天只做热品——这等于半年亏损。必须做季节菜单：夏天绿豆沙+冰银耳+杨枝甘露，冬天姜撞奶+热银耳羹+红豆沙。**不做季节切换=冬天关门。**

### ❌ 坑5：面积太大员工太多

糖水店需要堂食区但不能太大。20-30平足够小店模式。你开60-100平需要4名员工，月人工成本1.2-1.6万，日销360元根本不够。**面积越大亏越多。**

### ❌ 坑6：加盟连锁品牌

苗雨加盟麦记牛奶投入73万，前2月赚10万第3月就亏，门店经常被罚钱。加盟糖水铺的稽查督导比奶茶更严格，管理难度更高。"加盟商都知道这是一门投入复杂且不确定强的生意。"

## 九、趋势信号

### 正向信号

- 🟢 糖水契合"轻养生"趋势——桃胶、银耳等食材自带健康属性
- 🟢 广东/南方市场有深厚的糖水消费文化基础
- 🟢 赵记传承打破"糖水不出两广"魔咒——650店开到全国
- 🟢 糖水客单价18-28元比奶茶高——单碗利润空间不错
- 🟢 亲子家庭消费场景——糖水是"家庭分享型"消费

### 负向信号

- 🔴 独立糖水店日均客流20人——远低于奶茶店100人
- 🔴 消费频次极低——"偶发性解馋消费"
- 🔴 连锁品牌疯狂扩张降维打击——麦记640家、糖叙326家
- 🔴 价格战：30-40元→16-19元
- 🔴 奶茶品牌将糖水"杯装化"分流核心客源
- 🔴 冬天生意入冬——北方市场更惨
- 🔴 备料3.5小时+SKU繁杂=运营极其繁琐
- 🔴 糖水很难复制奶茶的辉煌——品类基因缺陷
- 🔴 "16万积蓄全亏光"不是个例

## 十、进阶路线

### Level 1：私域/社区糖水验证（0-1个月）

- 不开店，先在小区微信群卖糖水
- 每天只做3-5种糖水，精简备料
- 周末在小区门口摆摊/取货点
- 投入0-5000元（基础器具+食材）
- 目标：日稳定10碗以上
- 如果连10碗都卖不到，别往下走

### Level 2：精简菜单小店（1-3个月）

- 只做8-10种糖水（春夏各5种轮换）
- 选社区底商20-25平小铺位
- 简装+二手设备+夫妻店模式（不招员工）
- 建立微信私域客户群
- 日目标：20碗以上

### Level 3：季节菜单+外卖优化（3-6个月）

- 做好季节菜单切换（冬夏各5种核心产品）
- 优化外卖包装（碗装密封防洒）
- 做下午茶时段推广（写字楼白领）
- 日目标：30-50碗

### Level 4：差异化+融合模式（6-12个月）

- 像赵记传承的"融合店"模式——糖水+面食+蛋挞+炸鸡
- 增加冬季品类（关东煮、热小吃）对抗淡季
- 小红书/抖音内容营销（糖水颜值高天然好看）
- 做自有品牌特色产品（别抄麦记的木薯大满贯）

### Level 5：稳定经营+区域品牌（1年以上）

- 精简SKU到极致效率（标准化备料流程）
- 可能考虑2-3家小店（社区+学校各一家）
- 做区域口碑品牌（绝不做商圈加盟）
- 或者回归社区私域模式——低成本才是糖水生意的出路
