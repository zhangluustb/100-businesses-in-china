# 100个低成本生意商业秘籍项目

> 数据来源：小红书UGC内容（博主+评论区真实对话），知乎+新闻补充验证
> 创建日期：2026-05-17
> 目标：为每个行业生成一份接地气的大白话赚钱秘籍（基于真实UGC从业者分享）
> 技能：使用 business-analyzer skill（v2.0）+ xhs-explore skill 搜索小红书

## 项目工作流

1. **小红书搜索**：使用 xhs-explore skill 在小红书搜索该行业的UGC内容
2. **详情+评论提取**：获取高互动笔记详情，重点提取评论区从业者真实声音
3. **补充搜索**：知乎/新闻验证数据
4. **大白话总结**：张雪峰风格，直接、有态度、引评论
5. **输出**：research.md（原始数据）+ summary.md（大白话秘籍）

## 秘籍生成规范

每个行业的秘籍文档结构：

```
businesses/
  001_煎饼果子摊/
    research.md    # 搜索原始数据（博主观点、视频链接）
    summary.md     # 赚钱秘籍总结
```

### 秘籍文档模板

```markdown
# [行业名称] 赚钱秘籍

## 一、行业概览
- 启动资金：
- 月均营收范围：
- 毛利率：
- 回本周期：

## 二、核心赚钱观点（来自博主分享）
### 观点1：[标题]
- 来源：[平台-博主名-视频标题]
- 核心内容：

### 观点2：...

## 三、赚钱秘籍总结
### 1. 选址策略
### 2. 产品/服务策略
### 3. 定价策略
### 4. 获客策略
### 5. 成本控制
### 6. 避坑指南

## 四、关键数据
- 日均客流：
- 客单价：
- 日营业额：
- 月净利润：
```

## 技术实现

使用 skill `business-analyzer` 循环分析每个行业：
1. 搜索B站/抖音/小红书相关视频 
2. 提取核心赚钱观点
3. 归纳总结生成秘籍
4. 输出标准文档
